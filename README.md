# 🧮 Scientific Calculator

My first Python project. Built in about 3 hours while learning the language.

## What it does
- Basic operations: addition, subtraction, multiplication and division
- Scientific functions: square root, power and factorial
- Keeps a history of all operations
- Handles invalid inputs without crashing

## What I learned building this
- How to create and use functions with parameters
- How to build an interactive menu with while loops
- Error handling with try/except (the hardest part honestly)
- Using Python's math module

## How to run
```bash
python3 Calculadora.py
```

## Built with
- Python 3.14
- math module
